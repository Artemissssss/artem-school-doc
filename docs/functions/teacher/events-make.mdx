# Створення події

Коли ви увійшли в класну кімнату, як вчитель у вас буде кнопка Створення події

## Для чого вона? 

1. При натискані на неї, бот попросить вас надіслати текст події
2. Далі бот попросить надіслати дату у форматі дд.мм.рррр(приклад 11.09.2023)
3. Далі бот попросить надіслати час у форматі гг:хх(приклад 15:00)
4. Далі бот попросить надіслати для кого призанченна подія(можна написати в будь якому вигляді)
