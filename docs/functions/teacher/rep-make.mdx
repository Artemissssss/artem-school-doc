# Зробити оголошення

Коли ви увійшли в класну кімнату, як вчитель у вас буде кнопка Зробити оголошення

## Для чого вона? 

1. При натискані на неї, бот попросить вас надіслати текст оголошення
2. Після надсилання тексту, бот відправить цей текст, як оголошення всім учасникам кімнат
3. Після цього бот напише, що оголошення було надіслане
