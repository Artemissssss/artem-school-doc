# Створення матеріалу

Коли ви увійшли в класну кімнату, як вчитель у вас буде кнопка Створення матеріалу

## Для чого вона? 

1. При натискані на неї, бот попросить вас надіслати файл/текст
2. Напишіть текст або виберіть файл з пристрою та надішліть це в бот
3. Бот напише, що матеріали успішно завнатажені
