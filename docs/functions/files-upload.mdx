# Завантаження файла

Коли ви увійшли в класну кімнату у вас буде кнопка Файли

:::danger Обережно

**Не плутайте з кнопкою Завантаження файлів уроку**

:::

## Для чого вона? 

1. Спочатку вона попросить файл/зображення будь що для Завантаження
2. Виберіть з свого пристрою, що ви хочете надіслати та надішліть в бота
3. Далі бот напише, що файли успішно завнатажені
