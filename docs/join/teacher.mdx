# Як приєднатися вчителем

## Перевірте які дані вам надіслані для приєдання

Якщо вчитель вам надіслав запрошення типу:

https://t.me/artemisSchool_bot?start=ea-gKLjWZRNxcxurqWDZb

То це посилання для приєдання в клас


Якщо вчитель вам надіслав запрошення типу:

ea-gKLjWZRNxcxurqWDZb

То це id для приєдання в клас

### Приєдання посиланням

1. Якщо вам надіслали посилання, то перейдіть по ньому
2. У вас відкриється телеграм
3. Почніть чат з ботом і він вам напише, що ви успішно приєдналися в клас
![Фото](/img/image4.png "Фото")

### Приєдання id вчителем

1. Якщо вам надіслали id для приєдання, то переходите за посилання [`Посилання на бота`](https://t.me/artemisSchool_bot) та починайте чат
2. Далі бот надішле привітання та висвітіться три кнопки 
![Фото](/img/image1.jpg "Фото")
3. Натисніть на кнопку Приєднатися в клас, як вчитель
![Фото](/img/image1.jpg "Фото")
4. Далі бот попросить id учня, вам потрібно надіслати той що дав вчитель
5. Після надсилання id бот відповість, що вас додано в клас, якщо ви надіслали правильній id
![Фото](/img/image5.png "Фото")
