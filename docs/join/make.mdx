# Як створити клас?

1. Перейти за посиланням: [`Посилання на бота`](https://t.me/artemisSchool_bot) та розпочати чат
2. Далі бот надішле привітання та висвітіться три кнопки 
![Фото](/img/image1.jpg "Фото")
3. Натисніть на кнопку створити клас
![Фото](/img/image2.png "Фото")
4. Далі бот напише написати назву класа і ви напишіть назву нового класу
5. Далі бот надішле вам, що клас створений успішно
![Фото](/img/image3.png "Фото")

## Що робити з отриманими даними?

### Дані для учня

Надіслані ботом дані, потрібні для приєднання учня.

**Надішліть дані для приєднання учня, саме учням, щоб вони приєдналися в клас учнями**

В наступних туторілах написано, як приєднатися учнем

### Дані для вчителя

Надіслані ботом дані, потрібні для приєднання вчителя.

**Надішліть дані для приєднання вчителя, саме вчителям, щоб учні не зайшли в клас, як вчителі**

В наступних туторілах написано, як приєднатися вчителем
